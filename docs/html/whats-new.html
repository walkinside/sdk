COMOS Walkinside APIs
What's New in 10.2 SP1

As this is a Service Pack release, it maintains compatibility guarantees relative to 10.2. You can find details of those guarantees in API Compatibility Policy.

General

  • Resolved issue 5 "Cannot find Walkinside SDK in example projects solution"

    Using SDK examples with "All Users" installation never worked and is problematic to support, so this release requires SDK developers to have "Just Me" COMOS Walkinside installation (or manually update assembly references and paths to their liking).

  • Example .sln files were split into 64-bit and 32-bit versions. Please use the one that matches your installed COMOS Walkinside version.

Viewer SDK

Notable Additions

  • IVRViewerSdkForPreview has a new method GetProjectSettings. It gives access to a settings storage, which is isolated inside a namespace from any settings of Walkinside or other plug-ins. Plug-in authors are encouraged to try it out as a possible future replacement of the deprecated IVRSettingsManager and let us know of any issues or suggestions.
  • IVRViewerSdkForPreview has a new method CreateGroup that creates an IVRGroup from multiple branches in a single call. This method is faster than calling IVRGroupBuilder.Add in a loop, especially for online projects.

Full details can be found in Viewer SDK API Reference.

Core SDK

Notable Additions

  • The VolumeManager was renamed to ShapeManager as it now allows to create 'flat' graphical entities like billboards and cross marks. Support for Axis-aligned bounding boxes, spheres as well as cones have been also added.
  • The LightManager has been introduced and allows to create multiple lights in the virtual environment very easily. 3 types of lights are supported (spot, point and directional).

Exporter SDK

Exporter SDK API has been in "preview" mode since around 2015, but never got wider traction among SDK developers. This is partly due to lack of documentation and design, but also due to being somewhat niche.

As there are no plans to improve this API towards "official" level of support and seeing how its relevance decreases thanks to new Builders added to standard Walkinside, it has been decided to officially discontinue Exporter SDK API in this release.

Virtual Room Client Library

Notable Changes

  • Add a new parameter peerTag to the JoinRoomAsync method of IDiscoveryClient interface. A peer provides the peerTag when connecting to a virtual training room; first peer to connect to the room tags the room with its tag. Only peers matching room's tag are admitted to join the room.

Full details can be found in Virtual Room Client Library API Reference.

Command-Line Interfaces (CLIs)

Notable Additions

  • New Builder Tool was added as a unified CLI for several Walkinside Builders (main exception being Builder for PDMS). Please find build.exe in your installed folder.

Notable Deprecations

  • MicroStation Exporter plug-in and PDS Exporter plug-in CLIs were deprecated. Please use the new Builder Tool instead.

Notable Removals

  • Builder for JT CLI was removed and its functionality superseded by new Builder Tool CLI.
