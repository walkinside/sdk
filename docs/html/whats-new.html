COMOS Walkinside APIs
What's New in 10.2 SP4

As this is a Service Pack release, it maintains compatibility guarantees relative to 10.2. You can find details of those guarantees in API Compatibility Policy.

Please note that this release is expected to be the last service pack before 10.3. When 10.3 is released, we will take the opportunity to remove any or all deprecated API members, according to the API Compatibility Policy.

Policy Change Notice

We would like to let you know in advance that our API Compatibility Policy is going to be revised in the next release. Our current policy was designed with much shorter release cycles in mind. In practice it forces us to keep irrelevant APIs for many years, slowing down innovation and new development.

The upcoming changes will address that and will allow us to have shorter deprecation and removal periods.

Until the new policy is in place we commit to the guarantees provided by the current version of the policy.
