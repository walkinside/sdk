COMOS Walkinside APIs
What's New in 10.2 SP3

As this is a Service Pack release, it maintains compatibility guarantees relative to 10.2. You can find details of those guarantees in API Compatibility Policy.

Viewer SDK

Notable Additions

  • IVRViewerSdkForPreview has a new method UnregisterForm that unregisters a form that uses a non-default constructor (constructor with arguments).
  • IVRViewerSdkForPreview has two new methods ConvertToCustomCS and ConvertFromCustomCS that convert internal 3D coordinates to/from custom coordinate system defined in the "World Coordinate" settings.

Notable Deprecations

  • VRProductInfo class was deprecated because it is not useful for SDK developers.

Full details can be found in Viewer SDK API Reference.

Data SDK

Notable Additions

  • IProject has a new method GetNamespacedSettings. It gives access to a settings storage, which is isolated inside a namespace from any settings of Walkinside or other plug-ins. This is a follow-up to the addition of similar API to Viewer SDK in the previous release.

Full details can be found in Data SDK API Reference.

Command-Line Interfaces (CLIs)

Notable Additions

  • New 4D Import Tool was added as a way to populate Walkinside project with 4D simulation data. Please find 4d.exe in your installed folder. You can find more information on the new 4D simulation feature in the Viewer user manual.
