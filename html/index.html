COMOS Walkinside APIs
Welcome!

In this package you will find documentation and examples for COMOS Walkinside 10.3 APIs.

See whats-new for notable API changes and additions.

Package Overview

There are several APIs available in this release:

  • Viewer SDK
  • Core SDK
  • Data SDK

The APIs are split into two categories: official and preview. Correspondingly, examples and API reference files are placed in official and preview subfolders of the package.

Support level and compatibility guarantees are different for official and preview APIs, see API Compatibility Policy for details.

To use provided example projects you will need Visual Studio 2012 or higher and COMOS Walkinside 10.3 64-bit or 32-bit installed in Just Me mode. Open .sln file in the examples folder to start.
