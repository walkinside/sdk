COMOS Walkinside APIs
What's New

10.3.5

  1. Some settings-related interfaces have been moved to Comos.Walkinside.Common.dll.
  2. Some url-related interfaces have been moved to Comos.Walkinside.Common.dll.
  3. IVRViewerSdk.MainActor was introduced to move and change the orientation of the avatar.
  4. Comos.Walkinside.BranchKind (DataSdk) and vrcontext.walkinside.sdk.VRBranchKind (ViewerSdk) have been merged into Comos.Walkinside.Common.Branches.BranchKind.
  5. WalkinsideSDK2 assembly was renamed to Comos.Walkinside.Viewer.

10.3.4

  1. IVRCamera.CastRay was removed in Viewer sdk.
  2. Viewer-sdk examples binaries are now packaged in Walkinside SDK package under "Desktop SDK/examples/Viewer-SDK/Binaries".
  3. VR3DControl.OnAction* events have been removed.
  4. viewerSdkPreview.FavoriteTagSource was renamed to viewerSdkPreview.FavoriteSearchSource.

10.3.3

  1. Adds the newly created assembly to your project references: Comos.Walkinside.Common.dll
  2. Some types have been moved from one namespace to another: try using Comos.Walkinside.Common or Comos.Walkinside.Viewer.
  3. Math types have been moved to Comos.Walkinside.Common.dll and have been renamed to:
    • Comos.Walkinside.Common.Math.Point3d
    • Comos.Walkinside.Common.Math.Vector2d
    • Comos.Walkinside.Common.Math.Vector3d
    • Comos.Walkinside.Common.Math.Vector4d
    • Comos.Walkinside.Common.Math.AxisAlignedBox
  4. vrcontext.walkinside.sdk.IVRLabelManager has been renamed to Comos.Walkinside.Viewer.ILabelManager
  5. Selection API has been revamped from scratch, please use IVRViewerSDk.Selection.
