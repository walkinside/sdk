vrcontext.walkinside.sdk Namespace

Welcome to API Reference of COMOS Walkinside Viewer SDK!

Here you will find public API that is provided by Viewer SDK. It enables development of Viewer plug-ins and Viewer applications.

Good starting points could be IVRViewerSdk and IVRPlugin.
