The IVRGroup type exposes the following members.

Methods
  NameDescription
ApplyColor
Apply Color to group.
ApplyLocalTransform
Apply a transform in order to translate/scale/rotate. The pivot of all transformation will be around the original center of the group.
ApplyStyle
Apply given VRStyle to group
ApplyWorldTransform
Apply a transform in order to translate/scale/rotate.
Hide
Switches Off in the 3D Window the geometry contained in the selection.
JumpTo
Jump to the selection representation. See JumpTo(UInt32).
ResetColors
reset group Color to original.
ResetStyles
Reset all styles applied to group.
SetTransform
Apply a VRMatrix in order to translate/scale/rotate group using a transformation matrix
Show
Switches On in the 3D Window the geometry contained in the selection.
