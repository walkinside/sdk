Remarks

Each branch is a named collection of geometrical primitives ("elements") and child branches. Non-root branches have one parent branch. Although it is possible to have more than one root branch in a branch hierarchy, this is not well-supported and is not recommended.

Most common branch hierarchies are "CAD" and "FRT".
