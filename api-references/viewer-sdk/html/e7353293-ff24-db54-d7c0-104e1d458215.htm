Type: IVRSmartObjectCategory
Existing category object, if project already has smart objects assigned to this category; else new category object.
Exceptions
ExceptionCondition
ArgumentNullException Thrown if categoryName is null.
ArgumentException Thrown if one of the following takes place: categoryName is empty; categoryName contains NUL character; categoryName is too long.
Remarks

Category names are non-empty case-sensitive strings, unique in the scope of project.

This method always returns a category object, even if project has no smart objects yet in this category. This is how new categories are meant to be created.
