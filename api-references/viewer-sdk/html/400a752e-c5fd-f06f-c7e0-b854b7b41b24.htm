Parameters

name
Type: SystemString
Name of the setting. Can be any non-null non-empty string, at most 347 characters, lowercase and without NUL characters.
defaultValue
Type: TValue

Default value if setting value has not been set.

New settings are written to the project only after you assign them a non-default Value. Until then they will always return the default value.

Type Parameters

TValue
Type of the setting value. Can be one of String, Boolean, Double, Single, Int32, VRVector3D or an Int32-based enum.

Return Value

Type: IVRSettingTValue
An object that can be used to read/write this project setting.
Exceptions
ExceptionCondition
ArgumentNullException Thrown if name or defaultValue is null.
ArgumentException Thrown if name is empty, contains uppercase characters, contains NUL characters or exceeds 347 characters. Also thrown if this setting is already registered for a different TValue type.
