Remarks

Calling this method produces the following effect:

  1. Current selection is changed to this branch;
  2. Camera is directed towards this branch's geometry;
  3. Collision is switched off;
  4. Camera flies directly to this branch's geometry and stops in front of it.

This method requires license feature "CAD Hierarchy Jump To And Highlight" to be available and JumpTo feature to be enabled.
