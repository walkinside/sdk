Remarks
This is useful for plug-in authors to avoid name collision with settings of Walkinside application and of other plug-ins.
Examples
Reading settings in a plug-in:
C#
Copy
IVRViewerSdk viewer = ...;
var myPluginNamespace = new Guid("..."); // Use some value generated in advance.
var settings = viewer.ForPreview.GetProjectSettings(myPluginNamespace);
var mySetting = settings.GetSetting("my setting", defaultValue: "...");
var myOtherSetting = settings.GetSetting("setting 2", defaultValue: 0);
...
viewer.UI.ShowInfo(mySetting.Value);
...
mySetting.Value = "new value";
It is also possible to subscribe to value change notifications:
C#
Copy
mySetting.ValueChanged += (sender, args) =>
{
    ...
};
