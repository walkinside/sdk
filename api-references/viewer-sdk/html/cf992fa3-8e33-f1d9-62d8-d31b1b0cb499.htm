IVRScene Properties

The IVRScene type exposes the following members.

Properties
  NameDescription
BranchManager
Returns the branch manager from the project this scene belongs to
FileName
Returns the absolute path to the file name of the scene
GlobalName
Returns the global scene name; it is unique for the current root project, even including all its children scenes/sub projects. Format follows by convention: project1_project2_..._projectN!scene Ex.: root_project2!cad_project is a scene named "cad_project" that was attached to project "project2" that was itself attached to the project "root".
ID
Returns the unique scene ID of this scene. this a unique in within all project
IsSystem
Is that scene a system scene (mandatory to run walkinside, independant of the other loaded projects or models) ?
LocalBBox
get the local bounding box of the scene
LocalName
Unique name of a scene/model within its parent project's context
LocalSceneID
return the local Scene Id within its own project
Matrix
get or set the scene matrix
Project
Returns the master project to which this scene belongs to
ReadOnly
Is the scene's data available in read-only mode ?
