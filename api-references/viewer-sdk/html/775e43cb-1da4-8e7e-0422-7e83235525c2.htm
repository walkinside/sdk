IVRNamespacedProjectSettings

Provides access to an experimental feature, which is project-level settings scoped by a namespace and isolated from all other project settings, including Walkinside standard settings.

This helps plug-in authors to store settings in Walkinside project that are specific to their plug-ins, without risk of name collision with settings that belong to other plug-ins or to Walkinside.
