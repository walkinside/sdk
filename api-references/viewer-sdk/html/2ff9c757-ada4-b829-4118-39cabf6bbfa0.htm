IVRPlugin Interface
Walkinside application plugins must inherit from IPlugin. IPlugin exposes 2 methods to create and destroy the plugin and 1 property used by walkinside engine to get the description of the plugin.

How to run this example

  1. Create a new .Net class library project.

    A class library does not have to be COM visible.

    -
  2. Attach the .Net component walkinsidesdk.dll as a reference.

    Attaching this component makes all the Walkinside SDK API visible to the class library you are creating. Right click "references" and select "add reference". Select the browse tab, and open the file "walkinsidesdk.dll" located in the walkinside 6.0.0.x installation directory.

    -
  3. Copy paste the example code in the class source file. -
  4. Set the build output path to the "Plugins" folder where walkinside is installed.

    If the walkinside installation folder need to stay as is, it is also possible to make just a manual copy of the walkinside installation folder to a working folder.

    -
  5. Set the debug executable application to the "walkinsidenet.exe" application. -

// The most basic example of a working plugin for walkinside.
// This module does not do anything realy. The module just shows a message
// when the plugin is created and destroyed.
public class WIPlugin : INativePlugin
{
    // Get the plugin descriptor of this WIPlugin_step1 walkinside module.
    public VRPluginDescriptor Description
    {
        get
        {
            VRPluginDescriptor desc = new VRPluginDescriptor();
            desc.ShortDescription = "Walkinside Plugin Step 1";
            return desc;
        }
    }

    // The method called by walkinside, immediatly after the plugin assembly is loaded in walkinside.
    // parameter viewer : The context of the viewer when plugin is created.
    //          returns : True if creation of plugin succeeded.
    public bool CreatePlugin(IViewer viewer)
    {
        string message = "Walkinside Plugin Step 1 : Initialised";
        Console.WriteLine(message);
        MessageBox.Show(message);
        return true;
    }

    // The method called by walkinside, just before the plugin is removed from walkinside environment.
    // parameter viewer : The context of the viewer when plugin is created.
    //          returns : True if destruction of plugin succeeded.
    public bool DestroyPlugin(IViewer viewer)
    {
        string message = "Walkinside Plugin Step 1 : Destroyed";
        Console.WriteLine(message);
        MessageBox.Show(message);
        return true;
    }
}
