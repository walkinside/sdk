IVRViewerSdkPreviewCoreSdkInitialized Event
Event raised when Core SDK is initialized and Core is ready to be used. This event is raised after a project is opened.
Remarks
It is good practice to listen to this event to restore Core SDK model as Core SDK is disposed when a project is closed and reinitialized when a new project is opened.
