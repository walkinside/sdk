Parameters

discoveryServerEndPoint
Type: System.NetIPEndPoint
The IPEndPoint of discovery server, which is the same to the training room IPEndPoint until virtual room server supports multiple virtual rooms. IPEndPoint
sendTimeout
Type: SystemTimeSpan
The same send timeout used by both discovery and training room protocols， which indicates the amount of time their underlying TcpClient will wait for a send operation to complete successfully. If this timeout is exceeded, the underlying TCP connection will be closed by its protocol library.
receiveTimeout
Type: SystemTimeSpan
The same receive timeout used by both discovery and training room protocols， which indicates the amount of time their underlying TcpClient will wait to receive data once a read operation is initiated. If this timeout is exceeded, the underlying TCP connection will be closed by its protocol library.
synchronizationContext
Type: System.ThreadingSynchronizationContext
Synchronization context on which events should be raised. Null means default context (same as new SynchronizationContext()).
