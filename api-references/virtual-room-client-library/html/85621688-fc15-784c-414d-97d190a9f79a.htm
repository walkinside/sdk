The ITrainingRoomClient type exposes the following members.

Events
  NameDescription
AvatarAdded
Raised after a newly added avatar joined a virtual room.
AvatarModelChanged
Raised when an avatar is changing its model.
AvatarRemoved
Raised after an avatar left a training room.
AvatarSizeChanged
Raised when an avatar is changing its size.
AvatarSkinChanged
Raised when an avatar is changing its skin.
AvatarStateUpdated
Raised when avatar state update message is received.
CommunicationErrorIssued
Raised when communication error occurs.
Disconnected
Raised when client is disconnected from server.
LoggedIn
Raised to the room participant after he successfully logged in to a virtual room.
LoggedOut
Raised to the room participant after he successfully logged out of a virtual room.
UserDefinedMessageReceived
Raised when a user-defined message is received.
