vrcontext.walkinside.sdk.Preview.Core Namespace

Welcome to API Reference of COMOS Walkinside Core SDK!

This API is in "Preview" phase, meaning it is experimental and subject to change. If you find it useful and would like to see it in next release, or if you have suggestions, please let us know.

Core SDK is intended to augment Viewer SDK API with custom model rendering capabilities.

Good starting points could be ModelManager and Model.
