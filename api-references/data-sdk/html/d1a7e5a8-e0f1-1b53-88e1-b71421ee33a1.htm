attributeData
Type: System.Collections.GenericIEnumerableKeyValuePairIBranch, IDictionaryIAttributeKey, Object

Sequence of data items describing which branches to update, which attributes to change and what attribute values to use.

If provided attribute value is null, that attribute will be removed from respective branch.

Exceptions
ExceptionCondition
ArgumentNullException Thrown if attributeData is null.
ArgumentException Thrown if one of the following takes place: one of the branches in attributeData is null; one of the branches in attributeData does not belong to this attribute source's project; attribute updates for one of the branches in attributeData is null; attribute key for one of the branch updates in attributeData does not belong to this attribute source (e.g., its attribute source kind is different or project is different); attribute value for one of the branch updates in attributeData is of type or format not supported by this attribute source (e.g. some attribute sources may not support floating-point values).
NotSupportedException Thrown if the project is opened in read-only access mode or attribute source does not support updates.
Remarks
Due to internal batching, using this method is normally several times faster than using SetAttribute(IBranch, String, Object) or RemoveAttribute(IBranch, String) in a loop.
