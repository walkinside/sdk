ITreeTDescendants Property
Returns all Children (directly or indirectly) of this element. The order of returned elements is not defined, only certainty is all returned elements will encounter this object when traversing the parent (Parent of its parents).

Examples
If the CAD hierarchy of the model has the following content.
Copy
- R
  +- n1
  +- n2
  +- n3
     +- n4
     +- n5
Then the following code will dump all names of the nodes accessible by R in the output window.
Copy
IBranch root = project.Branches.GetRootBranch(VRBranchMode.CAD);
foreach(IBranch branch in root.Descendants)
    Console.WriteLine(branch.Name);
Possible Result: n1 n4 n3 n2 n5
