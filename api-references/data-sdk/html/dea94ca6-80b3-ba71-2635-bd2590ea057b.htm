Comos.Walkinside Namespace

Welcome to API Reference of COMOS Walkinside Data SDK!

Warning: this API is in "preview" mode. It is not stabilized at the moment and may change incompatibly in future releases.

Here you will find public API that is provided by Data SDK. It enables development of tools and scripts for batch processing of Walkinside projects.

A good starting point is ProjectManager class.

Interfaces
  InterfaceDescription
IAttributeCollection
Generic attribute collection, like attributes attached to a branch or a search result.
IAttributeInstance
Describes an Attribute Value belonging to a specific Branch element. A Branch element is an element of the COMOS Walkinside CAD-Hierarchy or FRT.
IAttributeKey
Describes the unique name of an Attribute value. These names are typically shown in COMOS Walkinside in a database (PDMS,PDS) value window in the left column. If the properties Source and Parts are the same the instances are considered the same object.
IAttributeSourceKind
Describes various details of an attribute source kind, like composition of attribute keys.
IBranch
Represents a branch, which is a tree node in one of the project's breakdown structures, also known as branch hierarchies.
IBranchManager
Manages the collection of branches. Banch objects are elements displayed in the COMOS Walkinside CAD-Hierarchy window, and the FRT Window.
ICredentialProvider
Provides application with credentials when access to a protected resource fails. An example of protected resource is Walkinside Web Service used to access online models.
IEntity
Represents a data entity from entity storage.
IEntityManager
Manages access to data entities.
IOnlineProject
Represents extended version of IProject with operations specific to online projects.
IOnlineSettingManager
Represents configuration settings of an online COMOS Walkinside project.
IProject
COMOS Walkinside project information, used as main access point for all data manipulation.
IProjectAttributeSource
Describes the reference to a Data Source.
IProjectAttributeSourceManager
Attribute sources of a project.
IRedline
A redline object in a Walkinside project.
IRedlineCollection
Collection of IRedline objects.
IRedlineGroup
A group of redline objects in a Walkinside project.
IRedlineGroupManager
Collection of project's redline groups.
ISettingManager
Configuration settings of a COMOS Walkinside Project.
ISmartObjectCategory
A named category that groups one or more smart objects in a project.
ISmartObjectManager
Collection of project's smart objects and categories.
IStandaloneProject
Represents extended version of IProject with operations specific to standalone projects.
IStandaloneSettingManager
Represents configuration settings of a standalone COMOS Walkinside project.
ITreeT
Defines methods to manipulate generic Tree Collections.
IUriTemplate
A URI with placeholders.
IUriTemplateManager
Collection of project's URI templates.
IViewpoint
COMOS Walkinside viewpoint: an annotated position and direction of camera, serving as a 3D bookmark.
IViewpointManager
Collection of project's viewpoints.
