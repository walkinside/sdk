ITreeT Properties

The ITreeT generic type exposes the following members.

Properties
  NameDescription
Children
Exposes the enumerator, which can be used to iterate over all the elements belonging to this object.
DepthLevel
Gets the amount of parents that needs to be traversed for this node until no parrent is encountered any more. The root node has a DepthLevel of 0.
Descendants
Returns all Children (directly or indirectly) of this element. The order of returned elements is not defined, only certainty is all returned elements will encounter this object when traversing the parent (Parent of its parents).
HasChildren
Test if current object has oher elements attched. If this element has no children it is a leaf in the Tree graph. This method can be faster then testing if iterator returns content.
Parent
Gets the parent element of this Tree element.
