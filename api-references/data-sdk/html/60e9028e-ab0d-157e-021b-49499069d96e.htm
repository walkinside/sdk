Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type IBranchManager. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Exceptions
ExceptionCondition
ArgumentNullException Thrown if name is null.
ArgumentException Thrown if one of the following takes place:
  • name contains NUL character;
  • name is longer than 200 characters.
InvalidOperationException Thrown if branchKind is Cad. CAD branch hierarchy is immutable and cannot be changed.
ArgumentOutOfRangeException Thrown if branchKind is lower than Cad or higher than 10.
Remarks
There can be several roots of the same BranchKind, but this setup is not recommended.
