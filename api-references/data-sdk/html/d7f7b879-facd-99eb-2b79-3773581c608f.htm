Exceptions
ExceptionCondition
ArgumentNullException Thrown if branchCreationParams is null.
ArgumentException Thrown if one of the following takes place:
  • one of items is null;
  • some item's ParentReference references a key that does not belong to ReferenceKey of any preceding item (a "forward" reference, self-reference or just invalid reference);
  • some item's Name contains NUL character;
  • some item's Name is longer than 200 characters;
  • some item's ElementSources list contains null;
  • some item's ElementSources list contains a branch from different IProject (a "foreign" element source);
InvalidOperationException Thrown if some item indicates that it should be created in CAD hierarchy. CAD branch hierarchy is immutable and cannot be changed.
ArgumentOutOfRangeException Thrown if some item indicates that it should be created in branch hierarchy whose kind is lower than Cad or higher than 10.
NotSupportedException Thrown if the project is opened in read-only access mode.
Examples

This example creates a small tree of 3 FRT branches: root, child and grandchild.

C#
Copy
var rootParam = new BranchCreationParams(
    BranchKind.Frt, "Name for new FRT root");
var creationParams = new[]
{
    rootParam,
    new BranchCreationParams(
        "refkey1",
        rootParam.ReferenceKey,
        "Name for child branch"),
    new BranchCreationParams(
        "refkey2",
        "refkey1",
        "Name for grandchild branch"),
}
branches.Create(creationParams);
