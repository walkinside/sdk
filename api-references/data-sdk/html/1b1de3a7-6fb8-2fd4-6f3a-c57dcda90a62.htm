Remarks

Body may contain placeholders in a form of ${AttributeName}. For example, it can be a template of search engine query like https://duckduckgo.com/?q=${AttributeName}.

Data SDK does not process or interpret placeholders in any way. This is normally done by Viewer or other applications.
