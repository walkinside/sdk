Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type IBranch. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Exceptions
ExceptionCondition
ArgumentNullException Thrown if sourceBranch is null.
ArgumentException Thrown if sourceBranch belongs to a project different from this branch (elements can only be shared within same project).
InvalidOperationException Thrown if this branch is from CAD hierarchy and not from FRT. CAD branches are immutable and cannot be changed.
Remarks
After this method successfully completes, the branch will be referencing all elements that sourceBranch references.
