Exceptions
ExceptionCondition
ArgumentNullException Thrown if sourceBranches is null.
ArgumentException Thrown if one of the following takes place:
  • one of items in sourceBranches is null;
  • one of branches in sourceBranches belongs to a project different from this branch (elements can only be shared within same project);
InvalidOperationException Thrown if this branch is from CAD hierarchy and not from FRT. CAD branches are immutable and cannot be changed.
NotSupportedException Thrown if the project is opened in read-only access mode.
ObjectDeletedException Thrown if underlying branch has been deleted.
Remarks
After this method successfully completes, the branch will be referencing all elements that are referenced by each branch from sourceBranches.
