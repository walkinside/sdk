Properties
  NameDescription
AccessMode
What access is granted to this project.

Full path to project's .vrp file that is currently in use by the Data SDK.
ArchivePath
Full path to project's .vrz or .tlz file, if project is compressed; else null.
AttributeSources
Sources of branch attributes, like PDMS or PDS attributes.
Branches
Collection to all branches existing in this project.
Credentials
Entities
Project's entity storage.
FilePath
Full path to original project's .vrp file.
Name
Name of the project, not necessarily unique. The name of the project does not need to be related with the filename. This name is used in the graphical user interface of COMOS Walkinside to reference this project.
RedlineGroups
Project's redline groups.
Settings
Configurations of this project like rendering mode, etc...
SmartObjects
Project's smart objects and categories.
UriTemplates
Project's URI templates
