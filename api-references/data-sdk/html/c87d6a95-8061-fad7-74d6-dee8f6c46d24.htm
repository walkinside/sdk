Remarks

This id uniquely identifies branch within a single Walkinside project. It is allocated internally when branch is created, which happens during export from original CAD model or when FRT is created.

The value of id is implementation-defined. It is not guaranteed to remain the same if project is re-exported from original CAD source. This means that a piece of CAD geometry may be assigned a different branch id on each export.

If you need to match branches between different revisions of the same project, you should rely on something that will identify the branch in a stable manner. For some projects branch name or certain engineering attributes can serve that purpose. This is not universal and depends on specific CAD source and project's information management practices.
